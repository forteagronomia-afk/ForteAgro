# FORTEAGRO — DOCUMENTO MESTRE DE ARQUITETURA COGNITIVA

Este documento define **o DNA completo do aplicativo ForteAgro**.
Ele existe para garantir que **nenhum desenvolvedor, designer ou IA**
perca funcionalidades, caminhos, decisões, cálculos ou integrações.

---

## 1. PROPÓSITO CENTRAL

O ForteAgro é o **cérebro operacional contínuo da fazenda**.

Ele:
- pensa junto com o produtor
- analisa dados o tempo todo
- executa cálculos matemáticos contínuos
- explica decisões
- simula cenários
- orienta ações reais

Nunca é apenas:
- um app de registro
- um dashboard
- um chat isolado

---

## 2. MODELO MENTAL GLOBAL

Estado contínuo da fazenda:

Estado = Dados + Regras + Tempo + Histórico + Intenção Humana

Esse estado é recalculado constantemente.

---

## 13. REGRA DE OURO PARA DESENVOLVIMENTO

Toda funcionalidade deve responder:
1. Onde vive?
2. Qual motor usa?
3. Que decisão gera?
4. Como explica?
5. Como registra?

Se não responder, não entra no app.
